# Requerimientos Funcionales

## 1. Inicio de Sesión

RF-01: El sistema debe permitir el inicio de sesión mediante correo electrónico o número de cédula y contraseña.

RF-02: El sistema debe validar que las credenciales coincidan con un usuario registrado.

RF-03: El sistema debe mostrar mensaje de error si las credenciales son incorrectas.


## 2. Recuperación de Contraseña

RF-04: El sistema debe permitir ingresar correo o cédula para recuperación de contraseña.

RF-05: El sistema debe enviar un correo con enlace de restablecimiento válido por 15 minutos.

RF-06: El sistema debe incluir verificación audible para confirmar la solicitud.


## 3. Administrador

RF-07: El administrador debe poder visualizar un dashboard con métricas de:
- Número total de estudiantes
- Número total de programas activos
- Número total de usuarios registrados

RF-08: El administrador debe poder crear, editar y eliminar programas educativos.

RF-09: El administrador debe poder registrar docentes, terapeutas, encargados legales y estudiantes.


## 4. Docente / Terapeuta

RF-10: El docente debe visualizar listado de grupos asignados.

RF-11: El docente debe poder crear Planes Educativos Individualizados (PEI).

RF-12: El docente debe poder editar PEI existentes.

RF-13: El docente debe registrar observaciones y avances por estudiante.


## 5. Encargado Legal

RF-14: El encargado debe visualizar el progreso de su estudiante.

RF-15: El encargado debe consultar el detalle del PEI.

RF-16: El encargado debe recibir alertas cuando se registre un nuevo avance.
